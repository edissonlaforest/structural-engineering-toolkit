# structural-engineering-toolkit
Open-source Python toolkit for structural engineering calculations, design utilities, and automation based on ACI, Eurocode, ASCE, and AASHTO standards.
# Structural Engineering Toolkit

Open-source engineering tools for structural engineers.

## Features

- Beam Design
- Column Design
- Footing Design
- Wind Load
- Seismic Load
- Concrete Calculator
- Steel Calculator
- Unit Conversion

## Future Roadmap

- Eurocode
- ACI 318
- AASHTO
- ASCE 7
- BIM Automation
- AI Design Assistant
